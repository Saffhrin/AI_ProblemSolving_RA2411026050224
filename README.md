# AI Problem Solving RA2411026050224

## Problem Implemented
Map Coloring Problem (CSP)

## Description
This project solves the Map Coloring Problem using a Constraint Satisfaction Problem (CSP) approach with backtracking.

## Technologies Used
- Python
- Tkinter (GUI)

## How to Run

### Option 1: Run Locally (Tkinter GUI)
Go to MapColoring folder and run:
python main.py
